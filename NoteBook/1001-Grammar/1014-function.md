# 二、操作建议
* 所有变量(局部 and 全局)在使用前、都定义，而不是直接在第一次赋值时由python自己创建  
* 局部变量 和 全局变量 的名字不要一样、避免混淆
* 函数内部 的 全局变量 全部加global 关键字、不论是否可修改
