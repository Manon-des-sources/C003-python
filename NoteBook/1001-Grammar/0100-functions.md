---------------------- | - | -------------
 range(k, n)           |   | 创建列表为：[k, k+1, k+2, ... , n-1]、range(0, n) 可以简写为range(n)
 range(k, n, s)        |   | 创建列表为：[k+0\*s, k+1\*s, k+2\*s, ... , k+m\*s( < n)]
 range(n, k, -s)       |   | 创建列表为：[n-0\*s, n-1\*s, n-2\*s, ... , n-m\*s( > k)] 
 print                 |   | print(*object, sep=' ', end='\n', file=sys.stdout, flush=False)
